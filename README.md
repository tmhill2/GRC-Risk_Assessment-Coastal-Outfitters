# GRC Risk Assessment – Coastal Outfitters

## Project Overview

This project simulates a Governance, Risk, and Compliance (GRC) risk assessment conducted for a fictional outdoor retail company, Coastal Outfitters, with approximately 250 employees.

The purpose of this assessment was to identify organizational assets, evaluate cybersecurity threats and vulnerabilities, determine risk levels, and recommend mitigating controls.

**Disclaimer:**  
This project uses a fictional company scenario for educational and portfolio purposes.

---

## Methodology

This assessment followed a NIST-inspired risk assessment methodology:

1. Asset Identification
2. Threat Identification
3. Vulnerability Analysis
4. Risk Scoring
5. Risk Prioritization
6. Control Recommendations

Risk scoring formula:

Risk Score = Likelihood × Impact

Risk Levels:

- Low = 1–5
- Medium = 6–10
- High = 11–15
- Critical = 16–25

## Key Findings

Critical risks identified:

- Microsoft 365 phishing
- Active Directory credential theft
- Employee laptop malware
- Cloud storage data leakage

Recommended controls included:

- Multi-factor authentication (MFA)
- Security awareness training
- Role-Based Access Control (RBAC)
- Password policy improvements
- Backup protection strategies
